BOM(Browser Object Model)

window.open : 높이와 너비를 입력하여 새창을 띄움

setTimeout : 일정시간 후 함수 한번 실행 / id리턴

clearTimeout : setTimeout을 종료

setInterval : 일정 시간마다 함수를 반복 실행 / id리턴

clearInterval : setInterval 종료

onload : 화면에 요소들이 로드 완료되면 자동으로 실행되는 함수

screen : client 운영체제 화면속의 속성값을 갖는 객체

location : 브라우저 주소표시줄(URL)와 관련된 객체

navigator : 브라우저에 대한 정보
